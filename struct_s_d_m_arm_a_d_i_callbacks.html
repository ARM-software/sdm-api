AP accesses

The device parameter indicates the address of the AP to access. It can also be set to SDM_DefaultDevice, and the debugger will use a default AP. For ADIv5 systems, the AP address is an APSEL value in the range 0-255. For ADIv6 systems, the AP address is an APB address whose width depends on the target implementation.
